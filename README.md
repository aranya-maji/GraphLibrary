-- Graph Library --

This is a project that can create parent to child node relations for visual representation in C# .Net Framework.

Current version: v1

Developed:
1. Nodes visible
2. 4 directional support
3. Can be configured through database of choice for maximum flexibility
4. Pan movement done
5. Zoom in/out supported

To be developed:
1. Create lines between the nodes
2. Introduce Staggered mode for compact visualisation
3. Print feature to be implemented
